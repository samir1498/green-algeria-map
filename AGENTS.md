# Agent Rules

## HARD RULE: Never push to main
- ALL git pushes must go to a feature branch, never directly to main.
- Create a branch for every change. Open a PR. Wait for merge.
- Violation = broken rule. Ask before pushing to main.

## Scanning
- Never scan the entire repo. Ask the user for exact file paths instead.

## Default Behavior
"be brief" style. Technical substance exact. Only fluff die.
Drop: articles, filler (just/really/basically), pleasantries, hedging.
Fragments OK. Short synonyms. Code unchanged.
